QQBot 
------

基于 WebQQ 协议的 QQ 机器人。专为精英盒子设计。  

>精英盒子 QQ 群：61137226


安装运行
------
* Clone
* 安装 Coffeescript: `sudo npm install -g coffee-script`
* 安装依赖：`npm install`
* Make your own copy of `cp config.demo.yaml config.yaml`
* 修改配置： `vi config.yaml`
* Run `coffee main.coffee` to keep your bot alive!
